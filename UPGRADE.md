# Upgrade Path / 升级路径

## English
- Lightweight version: free / low-cost starter
- Full one-file version: richer templates, SOPs, FAQ, examples
- Combo packs: multiple agents for one workflow
- Custom version: industry adaptation, knowledge base, forms, web tools

## 中文
- 轻量版：免费 / 低价试用
- 完整单文件版：更多模板、SOP、FAQ、案例
- 组合版：多个 Agent 串成一条工作流
- 定制版：行业适配、知识库、表格、网页工具
