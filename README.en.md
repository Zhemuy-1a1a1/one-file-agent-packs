# One-File Agent Packs

This repo turns common real-world scenarios into **one-file agent packs**.

## Who this is for
- Builders who want ready-to-use agent starters
- People doing AI content, AI services, or AI monetization
- Anyone turning prompts into deliverable products

## Current lightweight open-source packs
- Xiaohongshu Operator
- Xiaohongshu Leads Agent
- Short Video Writer
- Customer Support Assistant
- Sales Follow-up Agent
- Quote Generator
- Coding Assistant
- Data Analyst
- Meeting Notes Agent
- Project Manager Agent

## How to use
1. Open any markdown file in `agents/`
2. Copy it into your AI agent system prompt / instructions
3. Start with the included input template
4. Adapt it to your business if needed

## Upgrade path
- Lightweight open-source version
- Full one-file version
- Combo packs
- Custom version
