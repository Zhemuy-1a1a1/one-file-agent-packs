# One-File Agent Packs / 单文件 Agent 包

Ready-to-use one-file agent packs for real-world work.
可直接导入使用的单文件 Agent 包，面向真实业务场景。

---

## English

### What is this?
This repo contains lightweight one-file agent packs.
Each pack is a single markdown file you can directly import into your own AI agent as:
- system instructions
- a prompt starter
- an agent knowledge base seed

### Why this exists
Most people do not need more prompt theory.
They need something they can directly import, adapt, and use.

So the structure here is simple:
- one scenario
- one agent pack
- one markdown file

### Current open-source packs
- Xiaohongshu Operator
- Xiaohongshu Leads Agent
- Short Video Writer
- Customer Support Assistant
- Sales Follow-up Agent
- Quote Generator
- Coding Assistant
- Data Analyst
- Meeting Notes Agent
- Project Manager Agent

### How to use
1. Open any `.md` file in `agents/`
2. Copy it into your AI agent system prompt / instructions
3. Start with the input template inside the file
4. Adapt it to your own workflow if needed

### What is included in each pack?
Usually a pack contains:
- role definition
- core capabilities
- input template
- output structure
- basic rules
- usage notes

### Upgrade path
- Lightweight open-source version → free / low-cost trial
- Full one-file version → paid standard delivery
- Combo packs → multi-agent workflow packs
- Custom version → industry adaptation, knowledge base, forms, web tools

### Notes
This repo contains the lightweight public layer.
The full commercial version includes deeper templates, SOPs, FAQ, examples, and combo packs.

## 中文

### 这是什么？
这是一个 **单文件 Agent 包** 仓库。
每个 Agent 都整理成一个独立 markdown 文件，可以直接复制到你自己的 AI Agent 中，作为：
- 系统提示词
- 提示词底稿
- Agent 知识起点

### 为什么做这个？
大多数人缺的不是更多 prompt 理论，
而是一个 **拿到就能直接用** 的东西。

所以这里把结构压得很简单：
- 一个场景
- 一个 Agent 包
- 一个 md 文件

### 当前开源的轻量版方向
- 小红书运营助手
- 小红书获客Agent
- 短视频创作Agent
- 客服回复助手
- 客服跟单Agent
- 报价单Agent
- 代码编程助手Agent
- 数据分析Agent
- 会议纪要Agent
- 项目管理Agent

### 如何使用
1. 打开 `agents/` 目录中的任意 `.md` 文件
2. 复制到你自己的 AI Agent 系统提示词 / Instructions 中
3. 按文件里的输入模板开始使用
4. 如果有自己的业务，再继续二改

### 每个 Agent 包通常包含什么？
通常会包含：
- 角色定义
- 核心能力
- 输入模板
- 输出结构
- 基础规则
- 使用说明

### 版本路径
- 轻量开源版 → 先体验 / 先引流
- 完整单文件版 → 标准版交付
- 组合版 → 多 Agent 串联工作流
- 定制版 → 行业字段、知识库、表格、网页工具

### 说明
当前这个仓库放的是轻量开源层。
完整版商业包会包含更完整的模板、SOP、FAQ、案例库和组合包。
