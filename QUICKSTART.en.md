## Get started in 3 minutes

### 1) Pick one agent pack
Go here and choose the pack closest to your task:
[agents/INDEX.md](./agents/INDEX.md)

Examples:
- Want Xiaohongshu content help → [Xiaohongshu Operator](./agents/zh-CN/小红书运营助手.md)
- Want customer support replies → [Customer Support Assistant](./agents/zh-CN/客服回复助手.md)
- Want project planning help → [Project Manager Agent](./agents/zh-CN/项目管理Agent.md)

### 2) Copy the whole markdown file
Do not over-optimize at first.
Copy the entire file.

### 3) Paste it into your AI agent instructions
This can be called:
- system prompt
- instructions
- custom agent description
- project-level prompt setup

### 4) Start with the built-in input template
If you are not sure how to ask, see:
- [Sample Inputs](./examples/sample-inputs.md)
- [Sample Outputs](./examples/sample-outputs.md)

### 5) Run a first usable version, then adapt
Do not aim for perfection on the first try.
Get a usable result first, then refine tone, business rules, and context.

## Beginner FAQ

### Q1: Can I use this if I don't have a special agent framework?
Yes, as long as your tool supports custom instructions or a system prompt.

### Q2: Should I really paste the full file?
Yes. That is the easiest way to start.

### Q3: What if the output does not sound like me?
Add 1-2 style constraints in your request, such as:
- make it more concise
- make it more conversational
- make it more sales-oriented

### Q4: What is the difference between this and the full version?
The open-source version is lighter and easier to try.
The full version is more suitable for real business use, with deeper templates, FAQ, SOPs, and examples.
