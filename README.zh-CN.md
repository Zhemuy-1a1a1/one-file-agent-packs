# 单文件 Agent 包

这是一个把常见业务场景整理成 **单文件 Agent 包** 的仓库。

## 适合谁
- 想快速搭自己的 Agent
- 想做 AI 内容 / AI 接单 / AI 变现
- 想把 prompt 整理成可交付产品

## 当前轻量开源内容
- 小红书运营助手
- 小红书获客Agent
- 短视频创作Agent
- 客服回复助手
- 客服跟单Agent
- 报价单Agent
- 代码编程助手Agent
- 数据分析Agent
- 会议纪要Agent
- 项目管理Agent

## 使用方法
1. 打开 `agents/` 目录
2. 选择一个 md 文件
3. 复制到自己的 Agent 系统提示词中
4. 按输入模板开始使用

## 升级路径
- 轻量开源版
- 完整单文件版
