## 一、资料包总览

# 电商运营Agent 资料包（标准版）

> 商品优化、活动策划、数据复盘一体化，偏执行型运营助手。

---

## 1. 适合谁

- 淘宝/拼多多/抖店商家
- 代运营团队
- 品牌电商负责人

## 2. 客户怎么输入

- 商品名称
- 类目
- 卖点
- 目标平台
- 当前问题
- 是否有数据

## 3. Agent 会输出什么

- 标题/卖点优化
- 活动建议
- 页面优化建议
- 数据复盘要点

## 4. 内置 skills 清单

### 商品诊断
- 用途：识别商品标题、卖点、主图、详情页问题
- 什么时候用：用户给商品信息
- 如何调用：先做诊断后给建议

### 卖点提炼
- 用途：把杂乱卖点提炼成可展示表达
- 什么时候用：用户要改商品文案
- 如何调用：输出标题、卖点、详情页模块

### 活动策划
- 用途：给活动节奏和促销方案
- 什么时候用：用户要做大促或日常活动
- 如何调用：输出活动方案清单

### 竞品对比
- 用途：提炼竞争对手卖点和缺口
- 什么时候用：用户需要差异化
- 如何调用：给对比维度和策略

### 复盘报告
- 用途：根据数据做简版运营复盘
- 什么时候用：用户有点击/转化/GMV 数据
- 如何调用：输出问题-原因-动作

### 审校优化
- 用途：检查话术空泛、活动无门槛、卖点不聚焦、结论不落地
- 什么时候用：交付前
- 如何调用：压缩成可执行建议

## 5. 推荐调用顺序
1. 先用核心技能做需求澄清 / 主结果生成
2. 再按场景调用扩展技能增强输出
3. 最后调用审校技能收口

## 8. 示例调用

**示例需求：** 帮我优化一款儿童保温杯的卖点，用于拼多多详情页和主图文案。

**内部处理建议：**

## 国内安装与增强说明

### 先说结论
- 这个 Agent 的**基础能力**优先靠当前单 md 本体运行
- 外部 Skills 属于**增强项**，不是国内客户的默认门槛
- 如果后续确实需要增强能力，优先使用：`https://skillhub.tencent.com/`

### 当前建议顺序
1. 先直接使用这个单 md Agent
2. 如果基础版已经够用，不需要额外安装 Skills
3. 如果确实要更强自动化 / 专业能力，再去 SkillHub 搜索对应 skill
4. 如果环境里还没有 SkillHub 商店，先看：`https://skillhub-1388575217.cos.ap-guangzhou.myqcloud.com/install/skillhub.md`

### 对外统一口径
- GitHub 不是国内用户的默认安装入口
- SkillHub 才是国内优先入口
- GitHub / `npx skills add ...` 仅作为备选路径保留
