# Public Release Scope / 公开边界

## English

### Included in the public repo
- Standard public agent packs
- README / quick-start / index / examples
- Public-facing branding and repo metadata
- Public-friendly sample structures and basic usage notes

### Not included in the public repo
- Low-cost optimized versions
- China-source optimized versions
- Custom delivery versions
- Sales SOPs / private conversion scripts / internal operation docs
- Internal test reports, validation notes, and unpublished business materials

### Positioning
The public repo is for:
- direct trial
- public sharing
- trust building
- lead generation

The commercial differentiation stays in:
- optimization
- deployment fit
- workflow customization
- maintenance and iteration

## 中文

### 公开仓库包含什么
- 标准公开版 Agent 本体
- README / 快速开始 / 索引 / 示例文件
- 面向公开传播的品牌素材与仓库元信息
- 适合公开展示的结构模板与基础使用说明

### 公开仓库不包含什么
- 低成本优化版
- 国内源优化版
- 定制交付版
- 成交 SOP / 私聊转化脚本 / 内部运营文档
- 内部测试报告、验证笔记、未公开业务材料

### 对外定位
公开仓库主要承担：
- 直接试用
- 公开传播
- 建立信任
- 引流获客

真正的商业差异点保留在：
- 优化能力
- 落地适配
- 工作流定制
- 持续维护与迭代
