| | 公开标准版 | 增强优化版 | 进阶定制版 |
|---|---|---|---|
| **定位** | 直接可用、复制导入 | 生产可用、降低成本 | 按业务深改、长期运营 |
| **目标用户** | 初次尝试、个人使用 | 真实业务、团队使用 | 企业客户、深度需求 |
| **适用场景** | 验证想法、学习参考 | 正式交付、运营使用 | 定制开发、长期陪跑 |

## 公开标准版（当前仓库）

**特点：**
- 单文件结构，复制即可用
- 通用场景，覆盖面广
- 基础输入/输出模板
- 适合验证想法、先跑起来

**包含：**
- Agent 本体（md 文件）
- 基础使用说明
- 简单输入/输出模板

**不包含：**
- 低成本优化
- 国内源适配
- 行业定制字段
- 长期记忆配置
- 交付 SOP

---

## 增强优化版

**特点：**
- 输出结构更稳定
- API 调用成本更低
- 国内模型/国内源适配
- 真实业务场景优化

**相比标准版新增：**
- 输入模板字段扩展
- 输出格式强化（表格、JSON、结构化）
- 常见场景 FAQ
- 基础错误处理
- 国内源调用示例

---

## 进阶定制版

**特点：**
- 按行业/业务流程深度改造
- 记忆策略配置（长期记忆、上下文管理）
- 工作流自动化（多 Agent 协作）
- 交付支持与迭代维护

**相比增强版新增：**
- 行业专属字段与知识库
- 多轮对话记忆配置
- 自动化工作流设计
- 交付文档与 SOP
- 定期迭代与维护

## 怎么选

| 你的情况 | 推荐版本 |
|---|---|
| 想先试试效果 / 个人使用 | 公开标准版 |
| 想正式用于业务 / 降低成本 | 增强优化版 |
| 想深度定制 / 长期运营 / 企业级 | 进阶定制版 |

## 常见问题

**Q: 公开标准版够用吗？**
A: 对于验证想法、个人使用、简单场景，够了。如果发现经常需要调很久才能达到满意效果，就可以考虑升级。

**Q: 增强优化版主要优化了什么？**
A: 主要是三块：① 输出结构更稳定 ② API 成本更低 ③ 国内源环境更友好。

**Q: 进阶定制版包括哪些？**
A: 按具体需求来定，通常包括：行业知识库、多 Agent 协作、自动化流程、长期记忆配置、交付培训与维护支持。
