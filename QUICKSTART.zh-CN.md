## 3 分钟上手

### 1）先选一个 Agent
到这里挑一个最接近你需求的：
[agents/INDEX.md](./agents/INDEX.md)

举例：
- 想写小红书 → [小红书运营助手](./agents/zh-CN/小红书运营助手.md)
- 想做客服回复 → [客服回复助手](./agents/zh-CN/客服回复助手.md)
- 想做项目拆解 → [项目管理Agent](./agents/zh-CN/项目管理Agent.md)

### 2）复制整个 md 文件
不要只复制一部分，先把整个文件复制进去。

### 3）粘贴到你的 Agent 指令区
常见名称可能叫：
- 系统提示词
- Instructions
- 自定义 Agent 说明
- 项目级 Prompt

### 4）按文件里的输入模板提问
如果你不知道怎么提，先看：
- [示例输入](./examples/sample-inputs.md)
- [示例输出](./examples/sample-outputs.md)

### 5）先跑出第一版，再根据你的业务改
不要一上来就追求完美。
先让它出一版可用结果，再调。

## 小白常见问题

### Q1：我没有自己的 Agent，也能用吗？
只要你现在使用的工具支持自定义 Instructions / 系统提示词，就能试。

### Q2：我是直接把整个文件扔进去吗？
对，先整份复制进去最省事。

### Q3：如果输出不够像我的风格怎么办？
先在输入里补 1-2 条要求，比如：
- 语气更口语化
- 更简洁
- 更适合卖货

### Q4：这和完整版有什么区别？
开源版更轻，适合先跑起来。
完整版会更适合真实业务使用，通常会补更多模板、FAQ、SOP、案例。
